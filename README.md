🛒 Superstore Sales Dashboard

Interactive Power BI dashboard analyzing sales, profit, and product categories from the sample Superstore dataset. 📊✨

🔑 Key Features

💰 Total Sales & Total Profit cards

📈 Sales Over Time line chart

🛍️ Sales and Profit by Category/Region bar charts

🎛️ Interactive slicers for Region, Product Category, and Date

🛠️ Skills Used

🖥️ Power BI Desktop

🔄 Power Query (data cleaning & transformation)

📐 DAX (measures and calculations)

📂 Files

Superstore_Sales_Dashboard.pbix → full interactive Power BI file

Superstore_Sales_Dashboard.pdf → static version of the dashboard for quick viewing

🚀 How to Use

Open the .pbix file in Power BI Desktop to interact with the dashboard.

Use slicers to filter by Region, Category, or Date.

Hover over charts to view tooltips with detailed metrics.
